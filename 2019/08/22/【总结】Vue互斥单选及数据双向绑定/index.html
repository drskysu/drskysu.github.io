【总结】Vue互斥单选及数据双向绑定

Aug 22, 2019

如果是组合使用来实现互斥选择的效果，就需要 v-model 配合 value 来使用

template:

<div id="app">
    <label for="js">js<input type="radio" id="js" v-model="picked" value="js"></label>
    <label for="html">html<input type="radio" id="html" v-model="picked" value="html"></label>
    <label for="css">css<input type="radio" id="css" v-model="picked" value="css"></label>
    <div>{{picked}}</div>
</div>

js:

var vm = new Vue({
        el: '#app',
        data() {
            return {
                picked: 'js'
            }
        },
        methods: {

        }
    })

数据 picked 的值与单选按钮的 value 值一致时，就会选中该项
