python 第二课

Aug 12, 2019

在windows系统上安装Python的方法还算简单, 比平时安装个软件稍稍麻烦一点. 进入Python的官方下载页面Python.org/download, 你就会看到一堆下载链接. 我们就选择python 2.7.5 windows X86-64 installer,(针对64位的操作系统, 不过现在基本上都是64位的操作系统了).

不过这边要特别说明下, 假如你用的是python3版本的, 那么这个教程不是特别适合你, 因为python3 改动了不少的地方.

下载之后, 就和安其他软件一样, 双击, 一路next, 然后复制你的安装路径, 这个在后面要用上.

安装结束后, 我们还要设置环境变量, 这个就要用上上一步复制的安装目录了, 这是什么东西暂时可以不用管, 跟着下面的操作步骤进行设置就好了: 点击
